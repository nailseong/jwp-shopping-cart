# 장바구니 🛒

장바구니 미션 저장소

## 기능 요구 사항

- JWT 라이브러리를 활용하여 액세스 토큰 발급 기능
- 발급한 토큰을 이용하여 로그인이 필요한 기능(회원 정보 조회/수정, 회원탈퇴) 요청 시 포함하여 보내고 이를 이용하여 기능이 동작하도록 구현
